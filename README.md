# Exercicio 8 - Lista de Tarefas com Componentes Separados
**Objetivos:**
Criar um sistema de listas de tarefas onde cada tarefa é um componente separado, recebendo props para exibir o nome da tarefa, e um botão de "Concluir" que altera o estado da tarefa.
